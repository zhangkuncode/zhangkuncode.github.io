下面介绍一下半加器与全加器的原理

半加器：可以处理两个二进制一位数的加法器  

X(a) : 表示a进制下的数X  
0 + 0 = 0(10) = 00(2)  
0 + 1 = 1(10) = 01(2)  
1 + 0 = 1(10) = 01(2)  
1 + 1 = 2(10) = 10(2)  

![half][half]  

全加器：可以处理三个 二进制 一位数的相加  
 
0 + 0 + 0 = 0(10) = 00(2)      
0 + 0 + 1 = 1(10) = 01(2)     
0 + 1 + 0 = 1(10) = 01(2)      
0 + 1 + 1 = 2(10) = 10(2)      
1 + 0 + 0 = 1(10) = 01(2)      
1 + 0 + 1 = 2(10) = 10(2)      
1 + 1 + 0 = 2(10) = 10(2)      
1 + 1 + 1 = 3(10) = 11(2)
